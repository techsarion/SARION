# Sarion — Site-wide Keyword Map

One target cluster per page. Primary = the head term the page should rank for; secondary/long-tail/semantic support topical depth and capture mid/low-funnel intent. Commercial-intent terms signal buying readiness.

---

## `/` — Homepage
- **Search intent:** Informational + commercial (brand/category discovery)
- **Primary:** agency management software
- **Secondary:** client management software for agencies · all-in-one agency software · agency operations software

- **Commercial-intent:** agency software free trial · all-in-one agency platform pricing
